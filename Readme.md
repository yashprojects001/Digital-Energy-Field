# ⚡ Digital Energy Field

A futuristic **interactive particle energy field** built using **HTML, CSS, and JavaScript**.
Move your cursor across the screen and watch glowing particles react, connect, and form a dynamic **digital plasma network**.

This project creates a visually immersive **cyber-style energy environment** where particles behave like a living electrical field.

---

## 🌌 Preview

An animated canvas filled with glowing particles that:

* float across the screen
* connect with nearby particles
* respond to cursor movement
* simulate an interactive **energy network**

---

## ✨ Features

### ⚡ Interactive Energy Field

Particles respond to cursor movement and are attracted toward it, simulating an energy pull.

### 🌐 Dynamic Particle Network

Particles connect with nearby particles to form glowing lines, creating a digital network effect.

### 🎇 Neon Glow Visuals

Particles emit soft neon glows using canvas shadows for a futuristic aesthetic.

### 🖱️ Real-Time Interaction

Mouse movement influences the behavior of particles in real time.

### 📱 Full-Screen Canvas

The animation automatically scales to fit the browser window.

---

## 🧠 How It Works

The system uses a **particle simulation** rendered on an HTML5 canvas.

1. **Particles Spawn**

   * Random positions
   * Random velocities

2. **Particle Movement**

   * Each frame updates particle positions.

3. **Mouse Interaction**

   * Nearby particles move slightly toward the cursor.

4. **Particle Connections**

   * If two particles are close enough, a glowing line connects them.

5. **Continuous Animation**

   * The animation loop updates and redraws the scene using `requestAnimationFrame()`.
